Because “somewhere in the garage” isn’t a location. Home inventory for your actual clutter—electronics, parts, homelab and all.

This README is also the [GitHub Pages](https://hwinther.github.io/ClutterStock/) home page. Other published docs: [OpenAPI / ReDoc](https://hwinther.github.io/ClutterStock/api/) under `/api/`, the [database schema (SchemaSpy)](https://hwinther.github.io/ClutterStock/database/) under `/database/`, [project Markdown docs](https://hwinther.github.io/ClutterStock/project-docs/) under `/project-docs/`, test coverage under [`/coverage-dotnet/`](https://hwinther.github.io/ClutterStock/coverage-dotnet/) and [`/coverage-node/`](https://hwinther.github.io/ClutterStock/coverage-node/), the [Playwright E2E Allure report](https://hwinther.github.io/ClutterStock/e2e-allure/) under `/e2e-allure/`, the [monocart E2E dashboard](https://hwinther.github.io/ClutterStock/e2e-report/) under `/e2e-report/` (with the latest [Lighthouse audit](https://hwinther.github.io/ClutterStock/e2e-report/lighthouse-home.html) attached), the [monocart browser-side V8 coverage](https://hwinther.github.io/ClutterStock/coverage-e2e/) under `/coverage-e2e/`, the [self-rendered status SVGs](https://hwinther.github.io/ClutterStock/badges/) under `/badges/` (also distributed as `ghcr.io/hwinther/clutterstock/badges-e2e:latest` for air-gapped consumers — see [`scripts/badges/`](scripts/badges/) for the renderer), and the [OWASP ZAP DAST report](https://hwinther.github.io/ClutterStock/security/) under `/security/`.
